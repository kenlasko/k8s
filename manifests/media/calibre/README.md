# Summary
This contains two applications:
- [Calibre Web](https://github.com/crocodilestick/Calibre-Web-Automated) - For managing ebooks
- [ShelfMark](https://github.com/calibrain/shelfmark) - For downloading ebooks

Books downloaded via ShelfMark are automatically available in Calibre Web. Books can then be added to users' shelves where they are automatically synced to the respective user's Kobo E-Reader
